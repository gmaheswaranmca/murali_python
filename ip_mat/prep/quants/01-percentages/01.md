# Percentages — Phase 1 Notes (Foundations)

---

# 1. What is Percentage?

Percentage means:

> "Per hundred"

The symbol `%` means "out of 100".

Examples:

- 50% means 50 out of 100
- 25% means 25/100
- 120% means 120/100

---

# 2. Percentage ↔ Fraction Conversion

## Basic Formula

```math
Percentage = Value / 100
```

---

## Common Fraction → Percentage Table

| Fraction | Percentage |
| -------- | ---------- |
| 1/2      | 50%        |
| 1/3      | 33.33%     |
| 1/4      | 25%        |
| 1/5      | 20%        |
| 1/8      | 12.5%      |
| 2/5      | 40%        |
| 3/4      | 75%        |
| 4/5      | 80%        |

---

## Fraction to Percentage

Multiply by 100.

Example:

```
(3/5) × 100 = 60%
```

---

## Percentage to Fraction

Remove `%` and divide by 100.

Example:

```
75% = 75/100 = 3/4
```

---

# 3. Decimal ↔ Percentage Conversion

## Decimal to Percentage

Multiply by 100.

Example:

```
0.45 × 100 = 45%
```

---

## Percentage to Decimal

Divide by 100.

Example:

```
35% = 35/100 = 0.35
```

---

# 4. Finding Percentage of a Number

## Formula

```
x% of y = (x/100) × y
```

---

## Examples

### Example 1

Find 25% of 200.

```
25% of 200 = (25/100) × 200 = 50
```

---

### Example 2

Find 12% of 50.

```
12% of 50 = (12/100) × 50 = 6
```

# 6. Mental Math Tricks

## Trick 1 — 10%

10% means divide by 10.

Example:

```math
10% of 450 = 45
```

---

## Trick 2 — 5%

5% means half of 10%.

Example:

```math
10% of 200 = 20
5% of 200 = 10
```

---

## Trick 3 — 25%

25% means divide by 4.

Example:

```math
25% of 120 = 30
```

---

## Trick 4 — 50%

50% means half.

Example:

```math
50% of 86 = 43
```

---

# 7. Important IPMAT Observation

These are same:

```math
50% of 20 = 20% of 50
```

Both equal 10.
