# Percentages — Phase 2 Notes (Increase & Decrease)

---

# 1. Percentage Increase

## Formula

```
New Value = Original Value + Increase
```

or directly

```
New Value = Original × (1 + Percentage/100)
```

---

## Example 1

A salary of 20000 is increased by 10%.

### Method 1

10% of 20000:

```
(10/100) × 20000 = 2000
```

New salary:

```
20000 + 2000 = 22000
```

---

### Shortcut Method

```
20000 × (1 + 10/100)
```

```
20000 × 1.1 = 22000
```

---

# 2. Percentage Decrease

## Formula

```
New Value = Original × (1 - Percentage/100)
```

---

## Example

Price of a product is 500.

It decreases by 20%.

```
500 × (1 - 20/100)
```

```
500 × 0.8 = 400
```

---

# 4. Successive Percentage Increase

When increase happens multiple times.

---

## Example

Population increases by 10% and then by 20%.

Initial population = 1000

### Step 1

After 10% increase:

```
1000 × 1.1 = 1100
```

### Step 2

After another 20% increase:

```
1100 × 1.2 = 1320
```

Final population:

```
1320
```

---

# 5. Shortcut for Successive Changes

## Formula

```
Net % Change = a + b + (ab/100)
```

Applicable when both are increases.

---

## Example

Increase by 10% and then 20%.

```
10 + 20 + (10×20)/100
```

```
30 + 2 = 32%
```

So total increase = 32%.

---

Verification:

```
1000 → 1320
```

Increase:

```
320/1000 = 32%
```

Correct.

---

# 6. Successive Decrease

## Example

Price decreases by 10% and then by 20%.

Initial price = 1000

### Step 1

```
1000 × 0.9 = 900
```

### Step 2

```
900 × 0.8 = 720
```

Final value:

```
720
```

---

## Shortcut Formula

For two decreases:

```
Net decrease = a + b - (ab/100)
```

Example:

```
10 + 20 - (10×20)/100
```

```
30 - 2 = 28%
```

Net decrease = 28%.

---

# 7. Increase Followed by Decrease

Very important concept.

---

## Example

A price increases by 20% and later decreases by 20%.

Does it become same?

NO.

---

Assume original value = 100

### Increase 20%

```
100 × 1.2 = 120
```

### Decrease 20%

```
120 × 0.8 = 96
```

Final value = 96

Loss = 4%

---

## Shortcut Formula

If increase and decrease are same:

```
Net loss = (x² / 100)%
```

Example:

```
20² / 100 = 4%
```

---

# 8. Reversal Questions

---

## Example

Salary increased from 40000 to 50000.

Find percentage increase.

---

## Formula

```
Increase% = (Increase / Original) × 100
```

---

## Solution

Increase:

```
50000 - 40000 = 10000
```

Percentage increase:

```
(10000 / 40000) × 100
```

```
25%
```

---

# 9. Important Observation

Increase and decrease percentages are NOT symmetric.

---

## Example

Decrease from 100 to 80:

```
20% decrease
```

But increase from 80 to 100:

```
(20/80) × 100 = 25%
```

So:

* 20% decrease
* requires 25% increase to recover

Very important for aptitude exams.

# 10. Fast Mental Tricks

---

## Trick 1

Increase by 50%:

```
Multiply by 1.5
```

---

## Trick 2

Decrease by 25%:

```
Multiply by 0.75
```

or directly take:

```
3/4 of number
```

---

## Trick 3

Increase by 100%:

```
Value doubles
```

---

## Trick 4

```
Value becomes zero
```
